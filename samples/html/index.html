New Streams API

Browser samples demonstrating the runtime-agnostic Streams API

01 - Basic Stream Creation

Create streams with Stream.from(), Stream.pull(), arrays, and generators

02 - Consuming Streams

Read data with bytes(), text(), arrayBuffer(), and async iteration

03 - Transforms

Transform data with functions, generators, and stateful transforms

04 - Push-Based Streams

Create producer-driven streams with Stream.push() and writers

05 - Piping

Pipe streams through transforms to writer destinations with pipeTo()

06 - Branching

Multi-consumer patterns with broadcast() and share()

07 - Cancellation

Cancel streams with AbortSignal and handle cleanup

Running locally:

From the project root, run: node samples/html/server.mjs

Then open http://localhost:3000/samples/html/
